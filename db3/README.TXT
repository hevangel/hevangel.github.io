

                  IMPORTANT INFORMATION

Before using your ALS Version2.01r Disks, please note 
the following information before installing ALS v2.01r:

For 256K ALS v2.01r Operation:

ALS v2.01r runs with DOS version 2.xx if you have 256K
installed memory in your computer.  If you have a mimimum 
of 384K of installed memory, ALS v2.01r PLUS runs with DOS version
3.xx as well as version 2.xx. 

For 256K operation, we provide two files, CONFI256.SYS and
CONFI256.DB, on System Disk.  These two files set system
parameters for maximum overall performance of ALS v2.01r in a
256K environment.  Save the original CONFIG.SYS and CONFIG.DB
files (also on System Disk #1) to another disk, and then use the
DOS COPY command to copy CONFI256.SYS and CONFI256.DB to
CONFIG.SYS and CONFIG.DB respectively.
